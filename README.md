Website e-commerce

Các tính năng chính:

Đối với người dùng:
- Đăng ký, đăng nhập
- Xem danh sách, tìm kiếm sản phẩm, đánh giá sản phẩm
- Thêm sản phẩm vào giỏ hàng
- Thanh toán trực tuyến qua vnpay

Đối với admin:
- Dashboard thống kê doanh thu
- Quản lý sản phẩm, người dùng, đơn hàng

Công nghệ sử dụng
- Front-end: ReactJs
- Back-end: Node.js, Express.js
- Database: MongoDB
